Research

My passion is the design and implementation of practical tools that can help programmers to make more reliable and secure code. The goal of my research is to improve the process of verifying and testing complex software systems.

Software

Actively working on

  • SeaHorn : a verification framework for LLVM-based languages.
    SeaHorn has participated in SV-COMP, the competition on Software Verifiers:
    • SV-COMP'16 : one bronze medal in Termination.
    • SV-COMP'15 : one gold medal in Simple and two silver in ControlFlow and DeviceDrivers64.
  • Crab : a language-agnostic library for abstract interpretation.
  • Crab-LLVM : a static analyzer based on Crab for LLVM-based languages.
  • sea-dsa : a context-,field-, and array-sensitive heap analysis for verification of C/C++.

No longer working on

  • TRACER : interpolation-based symbolic execution tool for verification of C programs.
  • IKOS : Inference Kernel for Open Static Analyzers.
  • FTCLP: an interpolation-based solver for recursive Constrained Horn Clauses.
  • Wrapped Interval Analysis : a static analyzer of LLVM bitcode specialized for fixed-width integers.
  • Revenant : a string solver.
  • Covenant : a semi-decider for intersection of context-free languages.
  • The Ciao Preprocessor : a program analyzer for debugging, verification, and optimization for the Ciao programming language.
